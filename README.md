# Misuo-Sales-Analysis-Excel
Excel Sales Dashboard for Misuo Metal Furniture Company using Data Cleaning, Formulas, Pivot Tables, Charts, and Interactive Slicers.

## Project Overview

This project presents an interactive sales dashboard developed in Microsoft Excel for Misuo Metal Furniture Company.

The dashboard was created after cleaning and preparing the data using Excel, then building Pivot Tables, Pivot Charts, formulas, and interactive slicers to analyze sales performance and support business decision-making.

## Tools Used

- Microsoft Excel
- Data Cleaning
- Excel Formulas
- Pivot Tables
- Pivot Charts
- Interactive Slicers
- Dashboard Design

## Dashboard Features

- Total Sales
- Total Orders
- Average Selling Price
- On-time Delivery Rate
- Sales by Platform
- Orders by Area
- Best Seller Analysis
- Delivery Status Breakdown
- Best Color Sold

- Data Cleaning
- Data Analysis
- Dashboard Design

- Excel Visualization
